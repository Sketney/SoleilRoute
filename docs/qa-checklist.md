# Demo QA Checklist (Mobile)

Target widths: 360px, 414px, 768px.

- Login page: labels visible, button readable, no horizontal scroll.
- Register page: form fits viewport, password help/errors readable.
- Dashboard: left nav collapses/scrolls, header does not overlap content.
- Create trip (dialog): all fields visible, date pickers usable, submit reachable.
- Trip list: card text wraps cleanly, actions menu usable.
- Trip budget: charts do not overlap labels, add item form usable.
- Visa checker: dropdowns usable, result cards readable.
- Notifications: list scrolls, unread badge visible.
- Help page: accordion works, no clipping.
